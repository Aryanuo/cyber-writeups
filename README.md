# Cyber Writeups

This repository documents my cybersecurity learning journey through hands-on labs, vulnerable machines, and Capture The Flag (CTF) challenges.

The writeups in this repository focus on documenting the complete methodology used during each challenge, including reconnaissance, enumeration, vulnerability analysis, exploitation, privilege escalation, and key lessons learned.

My goal is to develop practical penetration testing and security assessment skills while maintaining clear and professional documentation. Whenever possible, writeups are structured similarly to real-world penetration testing reports.

Platforms and labs that will be documented include:

* TryHackMe
* Hack The Box
* VulnHub
* OverTheWire
* picoCTF

As I progress, this repository will grow to include more advanced challenges, techniques, and security concepts.

**Note:** All targets documented here are intentionally vulnerable lab environments, retired machines, training platforms, or authorized challenge environments used for educational purposes.
